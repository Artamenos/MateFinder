# MateFinder

MateFinder - fullstack-платформа для поиска тиммейтов в CS2. Сайт помогает игрокам создавать анкеты, указывать игровые навыки, смотреть статистику Faceit и отправлять приглашения другим игрокам.

## Для чего нужен проект

Проект решает частую проблему игроков CS2: сложно быстро найти подходящих тиммейтов с похожим уровнем игры, удобным временем онлайна, нужной ролью и общими картами. MateFinder собирает эту информацию в одном месте и делает поиск команды более удобным.

## Основные цели

- создать удобный каталог игроков;
- дать возможность заполнить подробную анкету;
- показывать игровые навыки и Faceit-статистику;
- реализовать систему приглашений в команду;
- продемонстрировать работу frontend и backend частей в одном проекте.

## Реализованный функционал

- регистрация и авторизация пользователей;
- личная анкета игрока;
- выбор роли: Rifler, AWPer, IGL, Support, Lurker;
- фильтрация тиммейтов по роли, уровню, карте и языку;
- карточки игроков с основной информацией;
- страница подробного профиля;
- обновление Faceit-статистики в mock-режиме;
- отправка, принятие и отклонение приглашений.

## Стек технологий

- Frontend: React, TypeScript, Vite
- Backend: Node.js, Express, TypeScript
- Database: SQLite
- ORM: Prisma
- Auth: JWT, bcrypt
- UI: CSS, lucide-react

## Запуск проекта

```bash
npm install
npm run db:generate
npm run db:push
npm run db:seed
npm run dev
```

После запуска:

- frontend: http://localhost:5173
- backend API: http://localhost:4000/api

## Примечание

Faceit-статистика сейчас работает через mock-сервис. Это позволяет демонстрировать полный сценарий проекта без внешнего API-ключа, но архитектура backend уже выделяет Faceit-интеграцию в отдельный сервис.
