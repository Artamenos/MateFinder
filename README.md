# MateFinder

MateFinder - fullstack-платформа для поиска тиммейтов и команд в CS2. Проект помогает игрокам находить подходящих напарников по уровню, ролям, картам, времени игры и игровым целям, а также собирать составы, отправлять приглашения и планировать пракки.

## Идея проекта

В CS2 часто сложно найти постоянную команду: игроки отличаются по уровню FACEIT, роли, прайм-тайму, пулу карт и ожиданиям от игры. MateFinder решает эту проблему как небольшая командная платформа: пользователь создает профиль, смотрит анкеты других игроков, подает заявки в команды, управляет составом и находит тренировочные матчи.

## Основные возможности

- регистрация и авторизация пользователей;
- личные профили игроков с ролью, уровнем FACEIT, картами, временем игры и описанием;
- импорт и хранение демо-профилей с FACEIT-данными;
- поиск тиммейтов по карточкам игроков;
- система команд: создание, редактирование, удаление, заявки на вступление и приглашения;
- управление составом команды: 5 игроков, тренер и замены;
- права капитана: редактировать команду, принимать заявки, менять капитана;
- раздел пракков с фильтрами по карте, времени, серверу, формату и названию;
- закрытая админ-панель для служебных функций;
- темный интерфейс в стиле CS2/FACEIT с фиолетовым акцентом.

## Стек

- Frontend: React, TypeScript, Vite, CSS;
- Backend: Node.js, Express, TypeScript;
- Database: PostgreSQL, Prisma ORM;
- Auth: JWT, bcrypt;
- Dev tools: npm workspaces, Docker Compose для локальной базы.

## База данных

Проект рассчитан на полноценную базу PostgreSQL. Локально ее можно поднять через Docker Compose, а при деплое подключить внешнюю PostgreSQL-базу на VPS или облачном сервисе через переменную `DATABASE_URL`.

База данных не хранится в GitHub-репозитории. В репозитории находятся схема Prisma, сиды и код приложения, а сами данные живут в PostgreSQL-сервисе.

## Структура

```text
client/  - frontend-приложение
server/  - backend API, Prisma-схема и seed-данные
```

## Быстрый запуск

```bash
npm install
docker compose up -d
copy server\.env.example server\.env
npm run db:generate
npm run db:push
npm run db:seed
npm run dev
```

Клиент открывается на `http://localhost:5173`, backend API работает на `http://localhost:4000`.
